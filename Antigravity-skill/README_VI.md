# Hướng dẫn Sử dụng Kỹ năng Chuyên gia Antigravity

Repository này được trang bị bộ **Kỹ năng (Skills)** chọn lọc để hỗ trợ bạn trong mọi giai đoạn của chu kỳ phát triển phần mềm.

## Phân loại Cụm Kỹ năng

| Nhóm Kỹ năng | Các Skill | Khi nào nên dùng |
| :--- | :--- | :--- |
| **1. Lên ý tưởng & Kế hoạch** | `@brainstorming`, `@concise-planning` | Khi bắt đầu dự án hoặc tính năng mới để làm rõ yêu cầu và lập danh sách các bước thực hiện cụ thể. |
| **2. Kiến trúc & Thiết kế** | `@architecture`, `@database-design`, `@api-design-principles`, `@frontend-design` | Khi thiết kế cấu trúc hệ thống, sơ đồ database, giao diện API (bao gồm SSE/WS), hoặc UI/UX cao cấp. |
| **3. Triển khai (Code)** | `@python-patterns`, `@react-patterns`, `@docker-expert`, `@git-pushing` | Trong quá trình viết code để áp dụng best practices cho FastAPI, React, tối ưu Docker và commit có cấu trúc. |
| **4. Chất lượng & Bảo mật** | `@test-driven-development`, `@lint-and-validate`, `@systematic-debugging`, `@seo-audit`, `@api-security-best-practices` | Khi viết test, fix bug bằng cách tìm nguyên nhân gốc, kiểm tra bảo mật API hoặc tối ưu hóa SEO. |

## Ví dụ Sử dụng Nhanh

- **Làm rõ ý tưởng thô**: `@brainstorming Mình muốn làm một app theo dõi thời gian thực...`
- **Thiết kế API chuẩn**: `@api-design-principles Thiết kế các endpoint SSE có kèm Keep-alive để cập nhật live.`
- **Fix bug logic khó**: `@systematic-debugging Tại sao kết nối WebSocket bị ngắt quãng định kỳ?`
- **Áp dụng chuẩn Python**: `@python-patterns Refactor đoạn code FastAPI này sử dụng Dependency Injection.`

*Xem chi tiết hướng dẫn kỹ thuật trong từng file `SKILL.md` tại thư mục `.agents/skills/`.*
