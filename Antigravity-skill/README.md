# Antigravity Expert Skills Guide

This repository is equipped with a curated set of **Antigravity Skills** to assist you through every stage of the software development lifecycle.

## Skill Categories

| Category | Skills | When to Use |
| :--- | :--- | :--- |
| **1. Planning & Ideas** | `@brainstorming`, `@concise-planning` | At the start of a project or feature to clarify requirements and create a step-by-step implementation plan. |
| **2. Architecture & Design** | `@architecture`, `@database-design`, `@api-design-principles`, `@frontend-design` | When designing system structures, database schemas, API contracts (including SSE/WS), or high-end UI/UX. |
| **3. Implementation** | `@python-patterns`, `@react-patterns`, `@docker-expert`, `@git-pushing` | During coding to apply best practices for FastAPI, React, Docker optimization, and structured commits. |
| **4. Quality & Security** | `@test-driven-development`, `@lint-and-validate`, `@systematic-debugging`, `@seo-audit`, `@api-security-best-practices` | When writing tests, fixing bugs using root cause analysis, auditing security, or optimizing for SEO. |

## Quick Usage Examples

- **Clarify a messy idea**: `@brainstorming I want to build a real-time tracking app...`
- **Design a robust API**: `@api-design-principles Design a set of SSE endpoints with Keep-alive for live updates.`
- **Debug a complex issue**: `@systematic-debugging Why is my WebSocket connection dropping periodically?`
- **Apply Python best practices**: `@python-patterns Refactor this FastAPI route using Dependency Injection.`
